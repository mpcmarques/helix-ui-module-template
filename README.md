# Helix React, Tailwind and Vite module template

Based on [mpcmarques/fivem-lua-react-tailwind-module](https://github.com/mpcmarques/fivem-lua-react-tailwind-module)
and [Simomagy/ns-htemplate](https://github.com/Simomagy/ns-htemplate)

A starter template for a Helix module using:

* React as the components library
* Tailwindcss as the design library
* Zustand as the global state manager

Also they come with prettier and eslint configured.


# How to use

## installing dependencies

Clone the repository, run yarn on ui folder

## development mode

### Run as a website

open a terminal on ui folder and run yarn then yarn start

## building

open a terminal on project folder and run yarn build (npm run build)
